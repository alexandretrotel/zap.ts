# Environment Variables

Zap.ts uses environment variables to configure your application for different environments (development, staging, production).

## Overview

Environment variables in Zap.ts follow a clear naming convention and are type-safe through validation schemas.

## Configuration

### Required Variables

Here are the essential environment variables you need to configure:

```dotenv
# Database
DATABASE_URL=your_production_database_url
DATABASE_URL_DEV=your_development_database_url

# Authentication
BETTER_AUTH_SECRET=your_auth_secret
BETTER_AUTH_URL=http://localhost:3000

# Email
RESEND_API_KEY=your_resend_api_key

# OAuth Providers (optional)
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```

### Optional Variables

Depending on which features you use:

```dotenv
# Analytics
POSTHOG_API_KEY=your_posthog_key
POSTHOG_HOST=https://app.posthog.com

# Payments
POLAR_ACCESS_TOKEN=your_polar_token
POLAR_WEBHOOK_SECRET=your_polar_webhook_secret

# AI/LLM
OPENAI_API_KEY=your_openai_key
ANTHROPIC_API_KEY=your_anthropic_key
```

## Environment Setup

### Development

1. Copy the example environment file:
   ```bash
   cp .env.example .env.local
   ```

2. Fill in your development values
3. Never commit `.env.local` to version control

## Validation

Zap.ts validates environment variables at startup using Zod schemas defined in `src/lib/env.ts`. This ensures:

- Required variables are present
- Values have correct format
- Type safety throughout the application

```ts
// Example validation
const envSchema = z.object({
  DATABASE_URL: z.string().url(),
  BETTER_AUTH_SECRET: z.string().min(32),
  // ...
})

export const env = envSchema.parse(process.env)
```

## Best Practices

- **Use descriptive names** - Make variable purpose clear
- **Group by feature** - Keep related variables together
- **Document requirements** - Note which variables are optional
- **Validate early** - Catch configuration errors at startup
- **Keep secrets secure** - Never expose sensitive data in client code
