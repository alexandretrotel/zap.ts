# Welcome to Zap.ts ⚡️

**Zap.ts** is a **Next.js** boilerplate designed to help you build applications _faster_ using a modern set of tools. It comes with best practices to build _excellent_ modern web applications.

## Why Choose Zap.ts?

- **Fast Setup** - Get started in minutes with a single command
- **Modern Stack** - Built with the latest tools and best practices
- **Highly Customizable** - Own your code and modify as needed
- **Well Documented** - Clear guides and examples for every feature
- **Type-Safe** - End-to-end type safety with TypeScript and Zod
- **Production Ready** - Battle-tested features for real applications

## Quick Start

Get started with Zap.ts in seconds:

<Tabs items={['npm', 'yarn', 'pnpm', 'bun']}>
<Tab value="npm">
```bash
npx create-zap-app@latest
```
</Tab>
<Tab value="yarn">
```bash
yarn create zap-app
```
</Tab>
<Tab value="pnpm">
```bash
pnpm create zap-app
```
</Tab>
<Tab value="bun">
```bash
bun create zap-app
```
</Tab>
</Tabs>

## What's Included?

Zap.ts comes packed with features to accelerate your development:

- **Authentication** - Authentication you *really own* with Better Auth
- **Database** - A type-safe ORM with Drizzle
- **UI Components** - Beautiful components with shadcn/ui
- **Payments** - Subscription handling ready to go
- **Analytics** - Built-in analytics integration
- **Email** - Transactional emails with Resend
- **And much more...**
